# OrbitTalk - Realtime Chat Application

OrbitTalk is a full-stack real-time chat application built using the MERN stack. It allows users to communicate instantly through a modern and responsive interface.

## Features

* User Authentication
* Real-time Messaging
* Online/Offline Status
* Responsive Design
* Secure Backend APIs
* Modern React UI

## Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS

### Backend

* Node.js
* Express.js
* MongoDB
* Socket.IO

## Installation

### Clone the Repository

```bash
git clone https://github.com/MrRaj-Varshney/OrbitTalk---Realtime-Chat-App.git
```

### Install Dependencies

```bash
npm install
```

### Run the Backend

```bash
cd backend
npm run dev
```

### Run the Frontend

```bash
cd frontend
npm run dev
```
